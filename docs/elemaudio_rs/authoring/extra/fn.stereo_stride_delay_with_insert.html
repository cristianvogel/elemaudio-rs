Expand description

Stereo stride delay with feedback insert loops.

Builds two independent mono insert delays (L/R) with per-channel tap names derived from props.fbtap: "{fbtap}:L" and "{fbtap}:R".

The insert closure is called twice — once per channel — with the feedback audio and a channel tag ("L" or "R") so the user can create per-channel keyed nodes inside the insert chain.

§Example

ⓘ
use elemaudio_rs::{el, extra};
use serde_json::json;

let [left, right] = extra::stereo_stride_delay_with_insert(
    json!({ "maxDelayMs": 1500, "transitionMs": 60, "fbtap": "fb" }),
    el::const_with_key("delay", 250.0),
    el::const_with_key("fb_amt", 0.5),
    |fb_audio, tag| {
        el::lowpass(
            el::const_with_key(&format!("insert_fc:{tag}"), 2000.0),
            el::const_(0.707),
            fb_audio,
        )
    },
    el::r#in(json!({"channel": 0}), None),
    el::r#in(json!({"channel": 1}), None),
);
